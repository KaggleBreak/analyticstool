This is the list of data set available fot this chapter, the way to retrieve them and possible polices for their usage:

(1) comtrade_trade_data_total_2003.csv

Starting from the 'Express Selection' interface (http://comtrade.un.org/db/dqQuickQuery.aspx, breadcrumb: comtrade.un.org/Annual data/Data Query/Express Selection) it is possible to get Comtrade data related to 'Total' transactions ('Import' and 'Export') among 'All' countries for a specific year, in our case 2003.
From the download page you can get the CSV file with all the data through the link 'Direct Download'.

how to use 'Express Selection': http://unstats.un.org/unsd/tradekb/Knowledgebase/How-to-use-the-Express-Selection-feature-in-the-Data-Query

(2) Country List: 

breadcrumb: comtrade.un.org/Annual data/Metadata & Reference/Reference Tables/UN Comtrade Reference Tables
direct link to the page: http://unstats.un.org/unsd/tradekb/Knowledgebase/UN-Comtrade-Reference-Tables see in the attachments: UN Comtrade Country List.xls 
direct link to the file: http://unstats.un.org/unsd/tradekb/Attachment392.aspx

(3) Product List:

breadcrumb: comtrade.un.org/Annual data/Metadata & Reference/Reference Tables/UN Comtrade Reference Tables
direct link to the page: http://unstats.un.org/unsd/tradekb/Knowledgebase/UN-Comtrade-Reference-Tables see in the attachments: "Commodity Classifications.rar" and inside it H1.txt
direct link to the file: http://unstats.un.org/unsd/tradekb/Attachment388.aspx
